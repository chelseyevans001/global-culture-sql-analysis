# Global Culture SQL Analysis

## Project Overview
This SQL project explores relationships between countries, landmarks, languages, and traditional foods.  
The goal is to demonstrate how relational databases can be used to analyze cultural data across different regions.

## Database Structure

The database contains four tables:

- countries – country information including continent and population
- landmarks – famous landmarks associated with countries
- languages – languages spoken in each country
- foods – traditional foods associated with each country

These tables are connected using the `country_id` field.

## SQL Skills Demonstrated

This project demonstrates the following SQL skills:

- CREATE TABLE
- INSERT INTO
- SELECT
- JOIN
- GROUP BY
- COUNT
- AVG
- ORDER BY

## Files Included

- schema.sql – creates the database and inserts data  
- queries.sql – SQL queries used for analysis
